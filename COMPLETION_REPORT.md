### 2. 自定义内容

- 修改联系方式 (电话、邮箱、地址)
- 更新工作室统计数据
- 添加真实作品信息
- 上传工作室照片

### 3. 配置表单提交

选择以下方案之一：
- **Formspree** - 最简单，无需后端
- **EmailJS** - 功能丰富，推荐
- **自建后端** - 完全控制

### 4. 部署上线

```bash
# 方案 1: Vercel (推荐)
npm install -g vercel
vercel

# 方案 2: Netlify
netlify deploy

# 方案 3: 自托管
npm run build
npm start
```

## 📝 自定义指南

### 修改工作室信息
1. **联系方式** - 编辑 `src/components/Contact.tsx`
2. **统计数据** - 编辑 `src/components/Studio.tsx`
3. **服务内容** - 编辑 `src/components/Services.tsx`
4. **作品信息** - 编辑 `src/components/Portfolio.tsx`

### 修改设计
1. **颜色主题** - 编辑 `tailwind.config.ts`
2. **字体样式** - 编辑 `src/globals.css`
3. **动画效果** - 编辑各组件的 className

### 上传图片
1. 创建 `public/images/` 文件夹
2. 放入图片文件
3. 在组件中引用: `<img src="/images/name.jpg" />`
